Midjourney V8 Parameters Cheat Sheet

📖 5 min read Published: March 18, 2026

Bookmark this page! Here's your complete reference for every Midjourney V8 parameter, with usage examples and recommendations.

🆕 New in V8

--hd (HD Mode)

Enables native 2048×2048 resolution rendering.

/imagine a detailed cityscape --v 8 --hd

Usage: Add when you want maximum quality. Uses ~2x GPU time.

📐 Aspect Ratio (--ar)

Sets the width-to-height ratio of your image.

--ar 1:1     Square (default)
--ar 16:9    Widescreen landscape
--ar 9:16    Vertical / phone wallpaper
--ar 4:3     Classic photo
--ar 3:4     Portrait photo
--ar 21:9    Ultra-wide cinematic
--ar 2:3     Poster format
--ar 3:2     Classic landscape photo

🎨 Stylize (--stylize or --s)

Controls how much Midjourney applies its own artistic style. Range: 0–1000.

--stylize 0      Minimal artistic interpretation
--stylize 50     Subtle styling
--stylize 100    Default balance
--stylize 250    Noticeably artistic
--stylize 500    Very artistic
--stylize 750    Strongly artistic
--stylize 1000   Maximum artistic flair

Recommendation: Start at default (100), increase for artistic shots, decrease for literal prompt matching.

🎲 Chaos (--chaos or --c)

Controls how much variation between the 4 generated images. Range: 0–100.

--chaos 0     Very similar outputs (default)
--chaos 25    Slight variations
--chaos 50    Moderate differences
--chaos 75    Significant differences
--chaos 100   Wildly different results

Recommendation: Use higher values when exploring ideas, lower when refining a concept.

🚫 Negative Prompt (--no)

Excludes specific elements from the generated image.

--no text
--no people, cars
--no watermark, signature, frame, border
--no blurry, low quality

Recommendation: Always use to remove unwanted artifacts like text and watermarks.

🔢 Version (--v)

Specifies which Midjourney model to use.

--v 8     Midjourney V8 (latest, recommended)
--v 7     Midjourney V7
--v 6.1   Midjourney V6.1

🌱 Seed (--seed)

Uses a specific seed for reproducible results. Range: 0–4294967295.

--seed 12345

Usage: Great for comparing prompt changes with consistent composition. Same seed + same prompt = same image.

⏹️ Stop (--stop)

Stops generation partway through for a less-finished look. Range: 10–100.

--stop 50     Stop at 50% completion
--stop 80     Stop at 80% (slightly rough)
--stop 100    Full generation (default)

Usage: Lower values create more abstract, painterly results.

🔮 Weird (--weird or --w)

Introduces unconventional, experimental qualities. Range: 0–3000.

--weird 250     Slightly unusual
--weird 1000    Notably strange
--weird 3000    Maximum weirdness

Usage: Perfect for surreal, avant-garde, or dreamlike imagery.

🔁 Repeat (--repeat or --r)

Runs the same prompt multiple times. Range: 2–40 (Standard), 2–10 (Basic).

--repeat 4    Generate 4 batches (16 images total)

🧮 Quality (--quality or --q)

Controls rendering quality/speed tradeoff.

--quality 0.25    Fast, rough preview (1/4 GPU)
--quality 0.5     Quick generation (1/2 GPU)
--quality 1       Default quality
--quality 2       Higher quality (2x GPU)
