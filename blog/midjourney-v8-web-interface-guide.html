How to Use Midjourney V8 on Web (No Discord Needed)

📖 14 min read Published: March 19, 2026 Updated: March 19, 2026

Great news for anyone who doesn't want to deal with Discord: Midjourney V8 is fully accessible through a dedicated web interface. The web app at alpha.midjourney.com offers a premium creative experience with features that actually surpass what's available on Discord.

In this guide, we'll walk you through everything you need to know about using Midjourney V8 on the web — from first login to advanced features you can't access anywhere else.

Getting Started with the Web Interface

Step 1: Create Your Account

  1. Visit alpha.midjourney.com
  2. Click "Sign Up" or "Sign In"
  3. You can create a standalone Midjourney account or sign in with Google, Discord, or Apple
  4. Choose a subscription plan (see our pricing guide)

💡 Note: You do NOT need a Discord account to use the web interface. A standalone Midjourney account works perfectly. However, if you already have a Discord-linked Midjourney account, you can sign in with Discord to access your existing history and subscription.

Step 2: Navigate the Web Interface

Once logged in, you'll see the main dashboard. Here's what each section does:

  • Imagine Bar (Top): The main prompt input field. Type your prompt here and press Enter to generate images.
  • Create Tab: Your primary workspace where you generate and view new images
  • Explore Tab: Browse trending images from the community for inspiration
  • Archive Tab: View, search, and organize all your past generations
  • Settings Panel: Adjust model version, parameters, and preferences

Creating Your First V8 Image on Web

Step 1: Select V8 as Your Model

Before generating, make sure V8 is selected:

  1. Look for the settings icon near the prompt bar (usually a slider icon)
  2. Find the "Model" or "Version" dropdown
  3. Select "V8" (it may be labeled as "V8 Alpha" during the testing phase)
  4. This setting will persist until you change it

Step 2: Write Your Prompt

Click on the Imagine Bar at the top and type your prompt. For example:

a magical library with floating books and warm candlelight, ancient wooden shelves stretching to infinity, dust motes dancing in golden light beams, photorealistic --ar 16:9

💡 Web Tip: On the web interface, you don't need to type /imagine or --v 8 — just type your prompt directly. The /imagine command is Discord-only, and V8 is automatically selected through your settings.

Step 3: Adjust Parameters Visually

One of the biggest advantages of the web interface: you can set parameters visually instead of typing them:

  • Aspect Ratio: Click the ratio selector to choose from presets (1:1, 16:9, 9:16, etc.) or enter custom values
  • Stylize: Use the slider to adjust from 0 to 1000
  • Chaos: Use the slider to set variation level
  • HD Mode: Toggle the HD switch for native 2K resolution
  • Quality: Choose between quality levels

Step 4: Generate and Review

  1. Press Enter or click the Generate button
  2. Your image grid will appear in real-time as V8 processes your prompt
  3. Wait for all four variations to complete (typically 30-60 seconds)
  4. Click on any image to view it at full resolution

Web-Exclusive Features

The web interface offers several powerful features not available through Discord:

1. Integrated Image Editor

After generating an image, you can edit it directly in the browser:

  • Inpainting: Select a region and re-generate just that area with a different prompt
  • Outpainting (Expand): Extend your image in any direction to create a larger canvas
  • Upscale Options: Choose between different upscale methods for your final output
  • Pan & Zoom: Navigate through variations and focus on specific areas

2. Image References & Blending

The web interface makes working with reference images incredibly intuitive:

  • Drag & Drop: Simply drag an image from your computer into the prompt area
  • Image Weight (--iw): Use the visual slider to control how much influence your reference has
  • Multi-Reference: Add multiple reference images to blend styles and concepts
  • Style Reference (--sref): Upload a style reference to maintain consistent aesthetics across generations

3. Project Folders & Organization

Keep your work organized with built-in project management:

  • Create folders for different projects or themes
  • Tag and categorize your generations
  • Search your entire archive by prompt keywords
  • Filter by date, aspect ratio, or model version
  • Star your favorite generations for quick access

4. Advanced Search

Unlike Discord's limited search, the web archive offers powerful filtering:

  • Search by any keyword in your prompts
  • Filter by specific parameters (e.g., show only --hd images)
  • Sort by date, likes, or relevance
  • View your generation history in grid or list view

5. Community Explore Page

The Explore tab is a goldmine for inspiration:

  • Browse trending images across all Midjourney users
  • See the exact prompts used for any public image
  • One-click to remix any community prompt with your own modifications
  • Filter by style, category, or trending period

Pro Tips for the Web Interface

  1. Keyboard shortcuts: Press Enter to generate, Esc to close the image viewer, and use arrow keys to navigate between results
  2. Quick remix: Click on any generated image, then click "Remix" to load its prompt back into the Imagine Bar with modifications
  3. Batch download: Select multiple images in your archive and download them all at once as a ZIP file
  4. Private mode: If you're on the Pro plan, enable Stealth mode directly from the web settings to keep your work private
  5. Multiple tabs: You can open multiple browser tabs to monitor several generations simultaneously
  6. Use bookmarks: Bookmark specific Explore pages or search results for quick future access

Common Web Interface Questions

Can I use the web interface on my phone?

Yes! The Midjourney web interface is fully responsive and works on mobile browsers. The experience is optimized for both landscape and portrait orientations. For the best mobile experience, use Chrome or Safari.

Do I need to install anything?

No. The Midjourney web interface runs entirely in your browser. No downloads, plugins, or extensions required. Just visit alpha.midjourney.com and sign in.

Can I use both web and Discord?

Absolutely! Your Midjourney account works across both platforms. Images generated on Discord appear in your web archive, and vice versa. You can switch between the two at any time.

Is the web interface slower than Discord?

No. Both interfaces use the same GPU infrastructure. Generation speed depends on your plan (Fast vs. Relax mode) and current server load, not which interface you use.

What browsers are supported?

The web interface works best on modern browsers including Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, and Brave. We recommend Chrome or Edge for the best experience.

Recommended Workflow for Web Users

Here's our recommended workflow for getting the most out of Midjourney V8 on the web:

  1. Plan your prompt: Use our Prompt Generator to build a structured prompt with the right parameters
  2. Set your parameters visually: Use the web interface sliders for aspect ratio, stylize, and chaos
  3. Generate in standard resolution first: Test your prompt at standard resolution to save GPU time
  4. Iterate and refine: Use the Remix feature to quickly modify prompts based on initial results
  5. Final render in HD: Once you're happy with the composition, enable --hd for the final 2K render
  6. Edit if needed: Use the integrated editor for inpainting or outpainting touches
  7. Organize: Move your best results into project folders for easy access
